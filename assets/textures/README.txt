# Game Textures

## Tileset
Place the following image files in this directory:
- overworld.png  (16x16 tileset - terrain)
- town.png       (16x16 tileset - buildings)  
- dungeon.png    (16x16 tileset - dungeon)
- objects.png    (16x16 tileset - objects)
